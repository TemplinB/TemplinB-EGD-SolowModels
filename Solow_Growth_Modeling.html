All data was taken from the Penn World Table (https://www.rug.nl/ggdc/productivity/pwt/). The World Bank was used for the savings rate which was taken as Gross Savings % GDP (https://data.worldbank.org/indicator/NY.GNS.ICTR.ZS). The most recent available data is from 2023. The depreciate rate I found was not given anywhere, so we are to assume 5%.

Important Note: Savings Rate is missing for Togo, Nigeria, and Liberia. To correct this, the last three known values for Togo are averaged (2017-2019) to calculate the current value. For Nigeria and Liberia, the median from all the countries in our report was taken. The same thing is done for Ghana and Liberia in finding alpha or capital share of income.

Import Note: All values are able to be adjusted, so if you found a different number I can put it into the model.

Consideration: I am not sure the savings rate is correct given the actual point on the graph, could be something to discuss or look into.
